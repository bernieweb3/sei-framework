# Narrative Layer Template

> Template for creating narrative framing that makes system operations feel like story experiences.

---

## Narrative Layer Overview

### Definition

The Narrative Layer is the story-like framing that transforms abstract system operations into engaging, contextual experiences. It doesn't change what the system does—it changes how users perceive and remember it.

### Purpose

```
Make "operating a system" feel like "exploring a world"
```

### Core Principle

```
Same functionality + Different framing = Different experience
```

|----|-------|

| Be specific | Be vague |

## Narrative Consistency Checklist

- [ ] All operations have narrative frames
- [ ] Environment has cohesive story setting
- [ ] Objects have narrative identities
- [ ] Actions have narrative descriptions
- [ ] Achievements have narrative titles
- [ ] Tone is consistent throughout
- [ ] Language guidelines are followed
- [ ] Microcopy uses narrative voice
- [ ] Story doesn't conflict with functionality
- [ ] Narrative enhances, not obscures, function

## Example: Complete Narrative Layer

### Project: DeFi Yield Farm

- Seed Vault → Store seeds (deposit)

- Deposit → "Plant seeds"

- Withdraw → "Gather your bounty"

**Achievements:**
- First Planting → "Green Thumb"
- 10 Harvests → "Master Gardener"
