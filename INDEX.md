## SEI-Lite interaction idea

```
Environment → Object → Overlay (Explain first, then Try) → Save → Export
```

---

## Terminology (short)

| Term | Meaning in SEI-Lite |
|------|---------------------|
| **Environment** | Theme + zones + allowed objects (max 8 on canvas) |
| **Object** | Spec’d block (text, button, form, etc.) |
| **Overlay** | Modal with **Giải thích** default, then **Thử ngay** |
| **Zone** | PROFILE, CORE, ACTION, TOOL, PROGRESSION |
