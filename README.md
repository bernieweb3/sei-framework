# SEI-Lite (Spatial Experience Interface — Lite)

This branch (`lite`) contains **`lite/`** as the product. For the **full SEI framework** (Foundation, Architecture, Implementation, Experience), use the **`main`** branch of the same repository: [github.com/bernieweb3/sei-framework](https://github.com/bernieweb3/sei-framework).

## What is SEI (in one minute)?

**Spatial Experience Interface (SEI)** orients people with **environment → object → overlay → action** instead of deep menu trees. **SEI-Lite** narrows that idea to **plain-language specs** for fast static sites and learners (including adults ~40–45).

```
Context first. Explain first. Then act.
```
