# Overlay guide / Hướng dẫn lớp phủ Explain-first

## Philosophy / Triết lý

**EN:** Understand first, act second.  
**VI:** Hiểu trước — làm sau.

## Tabs / Hai tab

### 🧭 Giải thích / Explain

**EN:** Short meaning, when to use, example, tips.  
**VI:** Giải thích ngắn, khi nào dùng, ví dụ, mẹo.

### ⚡ Thử ngay / Try

**EN:** Hands-on controls with live preview.  
**VI:** Thử chỉnh với xem trước trực tiếp.

## Why default matters / Vì sao mặc định quan trọng

**EN:** Many users (especially 40–45) want context before sliders and fields.  
**VI:** Nhiều người (đặc biệt 40–45) muốn hiểu ngữ cảnh trước khi chỉnh.

## Keyboard / Bàn phím

**EN:** Tab moves between controls; ESC closes; focus returns.  
**VI:** Tab để di chuyển; ESC để đóng; focus quay lại chỗ cũ.

## Saving / Lưu

**EN:** If you changed values, the app should ask before discarding.  
**VI:** Nếu đã chỉnh, ứng dụng nên hỏi trước khi huỷ.
