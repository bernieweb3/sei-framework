## How objects behave / Cách hoạt động

**EN:** Clicking an object opens an overlay: **Explain** (default) then **Try**.  
**VI:** Bấm vào đối tượng để mở lớp phủ: **Giải thích** trước, **Thử** sau.

## Instances / Số lượng

**EN:** Each spec sets `max_instances` to prevent clutter.  
**VI:** Mỗi đặc tả có `max_instances` để trang không quá rối.
