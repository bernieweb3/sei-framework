# Environment guide / Hướng dẫn môi trường

## What is an “environment”? / “Môi trường” là gì?

**EN:** A **theme + layout + allowed objects** package. It keeps your page coherent.  
**VI:** Một **gói giao diện + bố cục + danh sách đối tượng được phép** để trang của bạn đồng nhất.

## The four templates / Bốn mẫu

| Environment | EN “feels like” | VI gợi ý |
|-------------|-----------------|----------|
| Garden | Calm yard, soft story | Nhẹ nhàng, kể chuyện |
| Studio | Creative wall | Sáng tạo, portfolio nghệ thuật |
| Mini Shop | Small store | Bán hàng đơn giản |
| Portfolio | Crisp professional | Giới thiệu nghề nghiệp rõ ràng |

## Zones / Năm vùng

**EN:** PROFILE, CORE, ACTION, TOOL, PROGRESSION organize content so visitors aren’t overwhelmed.  
**VI:** Năm vùng giúp khách không bị ngợp.

## Limits / Giới hạn

**EN:** SEI-Lite allows **8** objects on the canvas—this is intentional for clarity.  
**VI:** Tối đa **8** đối tượng — cố ý để trang dễ hiểu.

## Choosing (decision tree) / Cách chọn

**EN:**

1. Selling something with SKUs or clear offer → **Shop**  
2. Showing creative projects → **Studio**  
3. Job hunting / consulting intro → **Portfolio**  
4. Gentle personal page → **Garden**

**VI:** Nếu bán → Shop; nếu showreel → Studio; nếu xin việc → Portfolio; còn lại → Garden.
