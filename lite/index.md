# SEI-Lite — AI agent entry

## What this is

SEI-Lite is a **specification-only** layer: **100% Markdown** under `lite/`. Implementations (React, Vue, static HTML, etc.) must be **generated by an AI or human** from these files—not shipped as source in this folder.

## Read order (recommended)

1. **Environments** — pick one template: [`01-ENVIRONMENTS/00-environment-index.md`](01-ENVIRONMENTS/00-environment-index.md)
2. **Objects** — compose from catalog: [`02-OBJECTS/00-object-index.md`](02-OBJECTS/00-object-index.md)
3. **Overlays** — Explain-first UX: [`03-OVERLAYS/00-overlay-system.md`](03-OVERLAYS/00-overlay-system.md)
4. **Export & deploy** — ship: [`04-EXPORT-DEPLOY/00-export-overview.md`](04-EXPORT-DEPLOY/00-export-overview.md)
5. **Validation** — before release: [`06-VALIDATION/00-validation-index.md`](06-VALIDATION/00-validation-index.md)
6. **User docs** — onboarding copy: [`07-DOCS/00-docs-index.md`](07-DOCS/00-docs-index.md)
